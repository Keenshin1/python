Módulo 11
Condicionais: if / elif / else
Tome decisões no seu programa com base em condições verdadeiras ou falsas.
Conceito
Todo programa precisa tomar decisões. Em Python, usamos if para executar um bloco de código apenas quando uma condição for verdadeira, elif para verificar uma segunda condição e else para o caso em que nenhuma condição anterior foi satisfeita.
A indentação (recuo de 4 espaços ou Tab) é obrigatória em Python — ela define quais linhas pertencem a cada bloco condicional.
Python
idade = int(input('Sua idade: '))

if idade >= 18:
    print('Maior de idade')
elif idade >= 16:
    print('Pode votar, mas não pode dirigir')
else:
    print('Menor de idade')
Operadores de comparação
== igual  |  != diferente  |  > maior  |  < menor  |  >= maior ou igual  |  <= menor ou igual
Operadores lógicos
Use and, or e not para combinar múltiplas condições em uma única expressão.
Operadores lógicos
nota = float(input('Nota: '))
faltas = int(input('Faltas: '))

if nota >= 7.0 and faltas <= 5:
    print('Aprovado!')
else:
    print('Reprovado.')
Atenção: = vs == Dentro de condições use sempre == (dois sinais de igual) para comparar. Um único = serve apenas para atribuir valores a variáveis.
Exercícios de fixação
01
Par ou ímpar
Usando o operador módulo com if/else
Leia um número inteiro e informe se ele é par ou ímpar. Dica: um número é par quando o resto da divisão por 2 é igual a zero (n % 2 == 0).
● Carregando Python...
▶ Saída
Solução
n = int(input('Digite um número: '))
if n % 2 == 0:
    print('O número {} é par.'.format(n))
else:
    print('O número {} é ímpar.'.format(n))
02
Classificador de notas
Usando if / elif / else encadeado
Leia a nota de um aluno (0 a 10) e exiba o conceito correspondente: A (9–10), B (7–8.9), C (5–6.9), D (abaixo de 5).
● Carregando Python...
▶ Saída
Solução
nota = float(input('Nota do aluno: '))
if nota >= 9:
    print('Conceito: A')
elif nota >= 7:
    print('Conceito: B')
elif nota >= 5:
    print('Conceito: C')
else:
    print('Conceito: D')
03
Aprovação com faltas
Usando operador lógico and
Leia a nota (0–10) e o número de faltas de um aluno. Ele é Aprovado se nota >= 6 e faltas <= 5. Se faltas > 5, é Reprovado por falta. Se nota < 6, é Reprovado por nota.
● Carregando Python...
▶ Saída
Solução
nota = float(input('Nota: '))
faltas = int(input('Faltas: '))
if nota >= 6 and faltas <= 5:
    print('Aprovado!')
elif faltas > 5:
    print('Reprovado por falta.')
else:
    print('Reprovado por nota.')
