Módulo 04
Operações matemáticas
Python como calculadora: soma, subtração, multiplicação, divisão e muito mais.
Conceito
Python suporta todas as operações matemáticas básicas. Como você viu no módulo anterior, todo valor tem um tipo — e input() sempre retorna uma string. Para fazer cálculos com o que o usuário digita, precisamos converter esse valor usando int() para inteiros ou float() para decimais.
int() e float() são funções de conversão. Você passa um valor dentro dos parênteses e recebe o mesmo valor em outro tipo:
Conversão de tipo
int('10')       # transforma a string '10' no número 10
float('3.14')   # transforma a string '3.14' no número 3.14
str(42)         # transforma o número 42 na string '42'
Na prática, combinamos int() com input() em uma só linha — o resultado de input() vai direto para dentro de int():
Operadores básicos
+ - * / // % **
Conversão de tipo
int() · float() · str()
Python
n1 = int(input('Primeiro número: '))
n2 = int(input('Segundo número: '))

print(n1 + n2)   # Soma
print(n1 - n2)   # Subtração
print(n1 * n2)   # Multiplicação
print(n1 / n2)   # Divisão real
print(n1 ** 2)   # Potência
print(n1 % n2)   # Resto da divisão
Exercícios de fixação
01
Soma de dois números
Conversão e operação básica
Leia dois números inteiros e mostre a soma deles.
● Carregando Python...
▶ Saída
Solução
n1 = int(input('Primeiro: '))
n2 = int(input('Segundo: '))
resultado = n1 + n2
print('A soma é: ' + str(resultado))
02
Dobro, triplo e raiz
Múltiplas operações sobre um número
Leia um número e mostre: seu dobro, seu triplo e sua raiz quadrada.
Dica: raiz quadrada = número elevado a 1/2, ex: n ** (1/2)
● Carregando Python...
▶ Saída
Solução
n = int(input('Digite um número: '))
print('Dobro: ' + str(n * 2))
print('Triplo: ' + str(n * 3))
print('Raiz: ' + str(n ** (1/2)))
03
Sucessor e antecessor
Operações simples com resultado formatado
Leia um número inteiro e mostre na mesma frase o número, seu antecessor (n - 1) e seu sucessor (n + 1).
● Carregando Python...
▶ Saída
Solução
n = int(input('Digite um número: '))
print('Número: ' + str(n) + ', antecessor: ' + str(n-1) + ', sucessor: ' + str(n+1))
04
Divisão inteira e resto
Operadores //, % e /
Leia dois números inteiros e exiba: a divisão inteira (//), o resto da divisão (%) e a divisão real (/) com 2 casas decimais.
Exemplo: 17 dividido por 5 → inteira: 3, resto: 2, real: 3.40
● Carregando Python...
▶ Saída
Solução
a = int(input('Dividendo: '))
b = int(input('Divisor: '))
print('Divisão inteira: ' + str(a // b))
print('Resto: ' + str(a % b))
print('Divisão real: ' + str(round(a / b, 2)))
