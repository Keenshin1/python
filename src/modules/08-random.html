Módulo 08
Módulo random
Gerando valores aleatórios e trabalhando com listas em Python.
Conceito
O módulo random permite gerar números aleatórios, sortear itens e embaralhar listas. As funções mais usadas são randint(), choice() e shuffle().
O que é uma lista?
Uma lista é uma forma de guardar vários valores em uma única variável. Em vez de criar aluno1, aluno2, aluno3... você guarda tudo dentro de colchetes separado por vírgulas. Cada valor ocupa uma posição que começa em 0.
Listas — introdução
frutas = ['maçã', 'banana', 'uva']

print(frutas[0])   # maçã   — primeiro elemento (índice 0)
print(frutas[2])   # uva    — terceiro elemento (índice 2)
print(len(frutas)) # 3      — quantidade de itens
Nota Neste módulo vamos usar listas de forma introdutória — apenas criar e passar para as funções do random. O estudo completo de listas (adicionar, remover, ordenar, percorrer) acontece no Módulo 14.
Módulo random
Com a lista criada, as funções do módulo random permitem sortear e embaralhar seus elementos de forma simples:
Módulo random
from random import randint, choice, shuffle

# Número aleatório entre 1 e 10
print(randint(1, 10))

# Sortear um item de uma lista
frutas = ['maçã', 'banana', 'uva']
print(choice(frutas))

# Embaralhar a lista (modifica a original)
shuffle(frutas)
print(frutas)
Exercícios de fixação
01
Sorteio de aluno
choice() com lista de nomes
Um professor precisa sortear 1 aluno entre 4 para apagar o quadro. Leia os nomes dos 4 alunos e exiba o escolhido.
● Carregando Python...
▶ Saída
Solução
from random import choice

a1 = input('Primeiro aluno: ')
a2 = input('Segundo aluno: ')
a3 = input('Terceiro aluno: ')
a4 = input('Quarto aluno: ')

lista = [a1, a2, a3, a4]
print('Aluno escolhido: {}'.format(choice(lista)))
02
Ordem de apresentação
shuffle() para embaralhar lista
O mesmo professor quer sortear a ordem de apresentação de trabalhos. Leia 4 nomes e mostre a ordem sorteada.

▶ Saída

from random import shuffle

a1 = input('Primeiro aluno: ')
a2 = input('Segundo aluno: ')
a3 = input('Terceiro aluno: ')
a4 = input('Quarto aluno: ')

lista = [a1, a2, a3, a4]

03
Dado virtual
randint() e indexação de lista
Simule o lançamento de 3 dados (cada um com valores de 1 a 6) usando randint(). Guarde os resultados em uma lista e exiba cada dado acessando pelo índice ([0], [1], [2]). Ao final, mostre a soma dos três resultados.
● Carregando Python...
▶ Saída
Solução
from random import randint

dados = [randint(1, 6), randint(1, 6), randint(1, 6)]
print('Dado 1: {}'.format(dados[0]))
print('Dado 2: {}'.format(dados[1]))
print('Dado 3: {}'.format(dados[2]))
print('Soma: {}'.format(dados[0] + dados[1] + dados[2]))
