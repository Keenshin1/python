Módulo 08
Módulo random
Gerando valores aleatórios e trabalhando com listas em Python.
Conceito
O módulo random permite gerar números aleatórios, sortear itens e embaralhar listas. As funções mais usadas são randint(), choice() e shuffle().
Listas são criadas com colchetes e armazenam múltiplos valores: [item1, item2, item3]. Você pode adicionar variáveis a uma lista e operar sobre elas.
Módulo random
from random import randint, choice, shuffle

# Número aleatório entre 1 e 10
print(randint(1, 10))

# Sortear um item de uma lista
frutas = ['maçã', 'banana', 'uva']
print(choice(frutas))

# Embaralhar a lista (modifica a original)
shuffle(frutas)
print(frutas)
Exercícios de fixação
01
Sorteio de aluno
choice() com lista de nomes
Um professor precisa sortear 1 aluno entre 4 para apagar o quadro. Leia os nomes dos 4 alunos e exiba o escolhido.
● Carregando Python...
▶ Saída
Solução
from random import choice

a1 = input('Primeiro aluno: ')
a2 = input('Segundo aluno: ')
a3 = input('Terceiro aluno: ')
a4 = input('Quarto aluno: ')

lista = [a1, a2, a3, a4]
print('Aluno escolhido: {}'.format(choice(lista)))
