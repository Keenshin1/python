Módulo 06
Importações e o módulo math
Usando bibliotecas prontas para cálculos matemáticos mais avançados.
Conceito
Python vem com módulos prontos que adicionam funcionalidades extras. Importamos com a palavra-chave import. O módulo math oferece funções matemáticas como raiz quadrada, seno, cosseno, tangente, etc.
Formas de importar
import math
math.sqrt(16)     # 4.0

from math import sqrt, trunc, hypot
sqrt(16)          # 4.0 (sem prefixo math.)
trunc(6.9)        # 6 (parte inteira)

from math import sin, cos, tan, radians
sin(radians(90))  # 1.0 (ângulo em graus → radianos)
Atenção com ângulos As funções trigonométricas do Python trabalham em radianos. Para passar graus, converta com radians(ângulo).
Exercícios de fixação
01
Hipotenusa
Teorema de Pitágoras com math.hypot
Leia o comprimento dos dois catetos de um triângulo retângulo e calcule a hipotenusa usando math.hypot().
● Carregando Python...
▶ Saída
Solução
import math

catOp = float(input('Cateto oposto: '))
catAd = float(input('Cateto adjacente: '))
hipo = math.hypot(catAd, catOp)
print('Hipotenusa: {:.2f}'.format(hipo))
02
Seno, cosseno e tangente
Trigonometria com conversão de ângulo
Leia um ângulo em graus e exiba seu seno, cosseno e tangente, cada um com 2 casas decimais.
● Carregando Python...
▶ Saída
Solução
from math import sin, cos, tan, radians

angulo = float(input('Ângulo em graus: '))
print('Seno: {:.2f}'.format(sin(radians(angulo))))
print('Cosseno: {:.2f}'.format(cos(radians(angulo))))
print('Tangente: {:.2f}'.format(tan(radians(angulo))))
