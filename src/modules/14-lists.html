Módulo 14
Listas
Armazene e manipule coleções de valores em uma única variável.
Conceito
Uma lista é uma coleção ordenada de valores. Cada valor ocupa uma posição (índice) que começa em 0. Você pode guardar inteiros, floats, strings ou até outras listas dentro de uma lista.
Criando e acessando listas
frutas = ['maçã', 'banana', 'laranja']

print(frutas[0])   # maçã   (primeiro elemento)
print(frutas[2])   # laranja (terceiro elemento)
print(frutas[-1])  # laranja (último elemento)
print(len(frutas)) # 3       (quantidade de itens)
Índices negativos lista[-1] acessa o último elemento, lista[-2] o penúltimo, e assim por diante. Muito útil para acessar o final sem precisar saber o tamanho exato da lista.
Métodos essenciais
Listas possuem métodos que permitem adicionar, remover e reorganizar elementos de forma simples.
Métodos de lista
numeros = [3, 1, 4, 1, 5]

numeros.append(9)     # adiciona 9 ao final → [3,1,4,1,5,9]
numeros.remove(1)     # remove a primeira ocorrência de 1 → [3,4,1,5,9]
numeros.sort()        # ordena em ordem crescente → [1,3,4,5,9]
print(sum(numeros))   # soma todos os elementos → 22
print(max(numeros))   # maior valor → 9
print(min(numeros))   # menor valor → 1
Percorrendo uma lista com for
O for percorre cada elemento de uma lista diretamente, sem precisar de índices. Use enumerate() quando precisar do índice junto com o valor.
Iterando sobre listas
notas = [8.5, 7.0, 9.2, 6.8]

# Percorrendo os valores
for nota in notas:
    print(nota)

# Percorrendo com índice
for i, nota in enumerate(notas):
    print('Aluno {}: {}'.format(i + 1, nota))
Exercícios de fixação
01
Média de uma lista
sum(), len() e iteração
Crie uma lista com 5 notas digitadas pelo usuário usando um laço for e append(). Ao final, calcule e exiba a média com 2 casas decimais.
● Carregando Python...
▶ Saída
Solução
notas = []
for i in range(5):
    n = float(input('Nota {}: '.format(i + 1)))
    notas.append(n)
media = sum(notas) / len(notas)
print('Média: {:.2f}'.format(media))
02
Maior e menor da lista
max(), min() e ordenação
Peça ao usuário 6 números inteiros, armazene-os em uma lista e exiba: o maior, o menor, a soma e a lista em ordem crescente. Depois, remova o menor número da lista com remove() e exiba a lista resultante.
● Carregando Python...
▶ Saída
Solução
nums = []
for i in range(6):
    n = int(input('Número {}: '.format(i + 1)))
    nums.append(n)
print('Maior: {}'.format(max(nums)))
print('Menor: {}'.format(min(nums)))
print('Soma: {}'.format(sum(nums)))
nums.sort()
print('Ordenada: {}'.format(nums))
nums.remove(min(nums))
print('Sem o menor: {}'.format(nums))
03
Filtrar aprovados
Listas + condicionais + funções
Crie a função filtrar_aprovados(notas, minimo) que recebe uma lista de notas e uma nota mínima, e retorna uma nova lista somente com as notas aprovadas (>= minimo). Leia 5 notas, chame a função com mínimo 6.0 e exiba os aprovados, reprovados e a média geral.
● Carregando Python...
▶ Saída
Solução
def filtrar_aprovados(notas, minimo):
    aprovados = []
    for nota in notas:
        if nota >= minimo:
            aprovados.append(nota)
    return aprovados

notas = []
for i in range(5):
    n = float(input('Nota {}: '.format(i + 1)))
    notas.append(n)

aprovados = filtrar_aprovados(notas, 6.0)
reprovados = []
for n in notas:
    if n < 6.0:
        reprovados.append(n)
media = sum(notas) / len(notas)
print('Aprovados: {}'.format(aprovados))
print('Reprovados: {}'.format(reprovados))
print('Média geral: {:.2f}'.format(media))
04
Inventário de loja
Listas paralelas + laços + formatação
Uma loja tem 4 produtos. Leia o nome e o preço de cada um e armazene-os em duas listas paralelas (nomes e precos). Ao final, exiba uma tabela formatada com todos os produtos e seus preços usando enumerate(), o produto mais caro e o valor total do estoque.
● Carregando Python...
▶ Saída
Solução
nomes = []
precos = []
for i in range(4):
    nome = input('Nome do produto {}: '.format(i + 1))
    preco = float(input('Preço: '))
    nomes.append(nome)
    precos.append(preco)

print('\n' + 30 * '-')
print('{:<20} {}'.format('Produto', 'Preço'))
print(30 * '-')
for i, nome in enumerate(nomes):
    print('{:<20} R$ {:.2f}'.format(nome, precos[i]))
print(30 * '-')

idx_caro = precos.index(max(precos))
print('Mais caro: {} (R$ {:.2f})'.format(nomes[idx_caro], precos[idx_caro]))
print('Total: R$ {:.2f}'.format(sum(precos)))
