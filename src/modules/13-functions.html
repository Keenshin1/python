Módulo 13
Funções com def
Agrupe blocos de código reutilizáveis em funções com parâmetros e retorno.
Conceito
Uma função é um bloco de código nomeado que pode ser executado ("chamado") quantas vezes quiser. Funções evitam repetição de código e tornam o programa mais organizado.
Definimos uma função com a palavra-chave def, seguida do nome e dos parâmetros entre parênteses. O return devolve um valor para quem chamou a função.
Definindo e chamando funções
def saudacao(nome):
    print('Olá, {}!'.format(nome))

saudacao('Ana')    # saída: Olá, Ana!
saudacao('Bruno')  # saída: Olá, Bruno!
Função com return
def soma(a, b):
    return a + b

resultado = soma(10, 5)
print(resultado)  # saída: 15
Boas práticas Nomeie suas funções com verbos que descrevam o que fazem: calcular_area(), converter_temperatura(), verificar_par(). Use underline para separar palavras.
Parâmetros com valor padrão
Parâmetros podem ter um valor padrão. Se o chamador não passar esse argumento, o valor padrão é usado automaticamente.
Valor padrão
def potencia(base, expoente=2):
    return base ** expoente

print(potencia(4))     # 16  (expoente padrão = 2)
print(potencia(2, 10)) # 1024
Exercícios de fixação
01
Calculadora de área
Criando e chamando uma função simples
Crie uma função chamada calcular_area(largura, altura) que retorna a área de um retângulo. Leia largura e altura com input(), chame a função e exiba o resultado.
● Carregando Python...
▶ Saída
Solução
def calcular_area(largura, altura):
    return largura * altura

l = float(input('Largura: '))
h = float(input('Altura: '))
area = calcular_area(l, h)
print('Área do retângulo: {} m²'.format(area))
02
Conversor de temperatura
Função reutilizável com return
Crie a função celsius_para_fahrenheit(c) que converte Celsius em Fahrenheit (fórmula: F = C * 9/5 + 32). Leia uma temperatura em Celsius e exiba o resultado em Fahrenheit com 1 casa decimal.
● Carregando Python...
▶ Saída
Solução
def celsius_para_fahrenheit(c):
    return c * 9 / 5 + 32

temp_c = float(input('Temperatura em Celsius: '))
temp_f = celsius_para_fahrenheit(temp_c)
print('{:.1f}°C = {:.1f}°F'.format(temp_c, temp_f))
03
Verificador de número primo
Combinando funções, for e condicionais
Crie a função eh_primo(n) que retorna True se n for primo e False caso contrário. Um número primo é divisível apenas por 1 e por ele mesmo. Leia um número, chame a função e exiba o resultado.
● Carregando Python...
▶ Saída
Solução
def eh_primo(n):
    if n < 2:
        return False
    for i in range(2, n):
        if n % i == 0:
            return False
    return True

num = int(input('Digite um número: '))
if eh_primo(num):
    print('{} é primo.'.format(num))
else:
    print('{} não é primo.'.format(num))
