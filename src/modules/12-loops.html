Módulo 12
Laços de repetição: for / while
Execute blocos de código múltiplas vezes de forma automática.
O laço for
O for em Python percorre uma sequência de valores. Combinado com range(), você controla quantas vezes o bloco se repete e com quais valores.
for com range()
# range(5) gera: 0, 1, 2, 3, 4
for i in range(5):
    print('Iteração', i)

# range(1, 6) gera: 1, 2, 3, 4, 5
for i in range(1, 6):
    print(i)

# range(0, 11, 2) gera: 0, 2, 4, 6, 8, 10
for i in range(0, 11, 2):
    print(i)
range(início, fim, passo) range(5) — de 0 até 4  |  range(1,6) — de 1 até 5  |  range(0,10,2) — pares de 0 a 8
O laço while
O while repete o bloco enquanto uma condição for verdadeira. É útil quando você não sabe de antemão quantas repetições serão necessárias. Cuidado: se a condição nunca ficar falsa, o programa entra em loop infinito!
while
contador = 1
while contador <= 5:
    print('Contagem:', contador)
    contador = contador + 1  # ou: contador += 1
Atenção Sempre garanta que a variável de controle do while seja atualizada dentro do loop, caso contrário o programa nunca para.
Exercícios de fixação
01
Tabuada
Usando for com range()
Leia um número inteiro e exiba sua tabuada completa (de 1 a 10), no formato: 5 x 1 = 5, 5 x 2 = 10, ...
● Carregando Python...
▶ Saída
Solução
n = int(input('Número para tabuada: '))
for i in range(1, 11):
    print('{} x {} = {}'.format(n, i, n * i))
02
Soma acumulada
Acumulando valores em uma variável
Leia um número inteiro N e calcule e exiba a soma de todos os números de 1 até N. Por exemplo, se N=5, a soma é 1+2+3+4+5 = 15.
● Carregando Python...
▶ Saída
Solução
n = int(input('Digite N: '))
soma = 0
for i in range(1, n + 1):
    soma = soma + i
print('Soma de 1 a {}: {}'.format(n, soma))
03
Contagem regressiva
Usando while com decrementação
Leia um número inteiro positivo e use while para exibir uma contagem regressiva desde esse número até 0, e ao final exiba "Fogo!".
● Carregando Python...
▶ Saída
Solução
n = int(input('Início da contagem: '))
while n >= 0:
    print(n)
    n -= 1
print('Fogo!')
04
Múltiplos de N
Usando range(início, fim, passo)
Leia um número inteiro N e um limite. Use range(N, limite + 1, N) para exibir todos os múltiplos de N até o limite (inclusive), um por linha. Ao final, exiba a quantidade de múltiplos encontrados e a soma deles.
● Carregando Python...
▶ Saída
Solução
n      = int(input('Valor de N: '))
limite = int(input('Limite: '))

soma       = 0
quantidade = 0
for mult in range(n, limite + 1, n):
    print(mult)
    soma       += mult
    quantidade += 1

print('Quantidade: {}'.format(quantidade))
print('Soma: {}'.format(soma))
