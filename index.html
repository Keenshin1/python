Módulo 01
Saída de dados com print()
A primeira coisa que todo programador aprende: exibir informações na tela.
Conceito
Em Python, usamos a função print() para exibir qualquer coisa na tela. Você passa o que quer exibir dentro dos parênteses.
Texto (chamado de string) precisa estar entre aspas — pode ser simples 'texto' ou duplas "texto".
Python
print('Olá Mundo!')
print("Python é incrível")
print(42)
print(3.14)
Saída do programa Olá Mundo!
Python é incrível
42
3.14
Dica Você pode usar \n dentro de uma string para pular linha: print('linha1\nlinha2')
Variáveis
Uma variável é um nome que guarda um valor na memória do programa. Pense nela como uma caixinha com um rótulo: você dá um nome à caixinha e coloca algo dentro.
Para criar uma variável, use o sinal = — que aqui significa "recebe", não igualdade matemática. O nome fica à esquerda e o valor à direita.
Python
nome = 'Tainá'        # a variável 'nome' guarda o texto 'Tainá'
idade = 28         # a variável 'idade' guarda o número 28
print(nome)        # exibe: Tainá
print(idade)       # exibe: 28
Atenção Nomes de variáveis não podem ter espaços nem começar com número. Use letras, números e underline (o traço embaixo _): nome_completo, nota1.

Módulo 02
Entrada de dados com input()
Como fazer o programa conversar com o usuário e capturar o que ele digita.
Conceito
A função input() pausa o programa e espera o usuário digitar algo. O que for digitado é retornado como uma string e pode ser guardado em uma variável.
Dentro dos parênteses de input() você pode colocar uma mensagem para orientar o usuário — isso é chamado de prompt.
Python
nome = input('Qual é o seu nome? ')
print('Olá, ' + nome + '!')
Atenção input() sempre retorna uma string, mesmo que o usuário digite um número. Para usar como número, é preciso converter com int() ou float().
Para combinar textos e variáveis no print(), use o operador + para concatenar (juntar) strings. Qualquer parte que não seja variável precisa estar entre aspas.
Concatenação com +
nome = input('Qual é o seu nome? ')
cidade = input('De onde você é? ')
print('Olá, ' + nome + '! Você é de ' + cidade + '.')

Módulo 03
Tipos primitivos e métodos de string
Entendendo os tipos de dados e explorando as capacidades das strings em Python.
Conceito
Todo valor em Python tem um tipo. Os principais são: str (texto), int (inteiro), float (decimal) e bool (verdadeiro/falso). Use type() para descobrir o tipo de qualquer valor.
Strings possuem métodos — funções que já vêm embutidas em todo texto. Pense assim: assim como um canivete suíço tem ferramentas dentro dele, uma string carrega suas próprias ferramentas. Para usar, escreva o nome da variável, um ponto, e o nome da ferramenta: variavel.metodo().
Métodos de string
texto = 'Python'

print(type(texto))          # descobre o tipo → <class 'str'>

# Métodos de verificação — retornam True ou False
print(texto.isupper())      # está tudo em MAIÚSCULAS? → False
print(texto.islower())      # está tudo em minúsculas? → False
print(texto.isalpha())      # contém só letras (sem número)? → True
print(texto.isnumeric())    # contém só números? → False
print(texto.isalnum())      # contém só letras e/ou números? → True
print(texto.istitle())      # cada palavra começa com maiúscula? → True

# Métodos de transformação — retornam o texto modificado
print(texto.upper())        # converte tudo para MAIÚSCULAS → PYTHON
print(texto.lower())        # converte tudo para minúsculas → python
print(texto.title())        # primeira letra de cada palavra em maiúscula → Python

# len() conta o número de caracteres
print(len(texto))           # quantos caracteres tem? → 6

Módulo 04
Operações matemáticas
Python como calculadora: soma, subtração, multiplicação, divisão e muito mais.
Conceito
Python suporta todas as operações matemáticas básicas. Como você viu no módulo anterior, todo valor tem um tipo — e input() sempre retorna uma string. Para fazer cálculos com o que o usuário digita, precisamos converter esse valor usando int() para inteiros ou float() para decimais.
int() e float() são funções de conversão. Você passa um valor dentro dos parênteses e recebe o mesmo valor em outro tipo:
Conversão de tipo
int('10')       # transforma a string '10' no número 10
float('3.14')   # transforma a string '3.14' no número 3.14
str(42)         # transforma o número 42 na string '42'
Na prática, combinamos int() com input() em uma só linha — o resultado de input() vai direto para dentro de int():
Operadores básicos
+ - * / // % **
Conversão de tipo
int() · float() · str()
Python
n1 = int(input('Primeiro número: '))
n2 = int(input('Segundo número: '))

print(n1 + n2)   # Soma
print(n1 - n2)   # Subtração
print(n1 * n2)   # Multiplicação
print(n1 / n2)   # Divisão real
print(n1 ** 2)   # Potência
print(n1 % n2)   # Resto da divisão

Módulo 05
Formatação de saída
Controlando casas decimais, alinhamento e como os dados aparecem na tela.
Conceito
O método .format() é uma forma mais organizada de montar strings com variáveis. Em vez de usar + e str(), você escreve {} dentro da string onde quer inserir um valor, e passa os valores dentro de .format():
Básico do .format()
nome = 'Tainá'
idade = 28

# com +  e str() — jeito antigo
print('Olá ' + nome + ', você tem ' + str(idade) + ' anos.')

# com .format() — mais limpo
print('Olá {}, você tem {} anos.'.format(nome, idade))
Dentro das chaves {} você também pode usar especificadores para controlar como os números são exibidos — útil especialmente para decimais:
Exemplos de formatação
n = 3.14159
nota = 8.5

print('{:.2f}'.format(n))      # 3.14 — 2 decimais
print('{:.0f}'.format(n))      # 3 — sem decimais
print('{:10.2f}'.format(n))    # "      3.14" — largura 10
print('{:1f}'.format(nota))    # 8.5

# f-strings (Python 3.6+) — alternativa moderna
print(f'Nota: {nota:.1f}')
Separador de linhas Você pode repetir um caractere com *: print(20 * '=') imprime 20 sinais de igual seguidos. Útil para criar bordas visuais.

Módulo 06
Importações e o módulo math
Usando bibliotecas prontas para cálculos matemáticos mais avançados.
Conceito
Python vem com módulos prontos que adicionam funcionalidades extras. Importamos com a palavra-chave import. O módulo math oferece funções matemáticas muito úteis no dia a dia: raiz quadrada, arredondamento, valor absoluto e a constante Pi.
Formas de importar
import math
math.sqrt(16)     # 4.0 — raiz quadrada

from math import sqrt, ceil, floor, trunc
sqrt(25)          # 5.0  — sem prefixo math.
ceil(4.2)         # 5    — arredonda para CIMA
floor(4.9)        # 4    — arredonda para BAIXO
trunc(6.9)        # 6    — parte inteira (descarta decimal)
O módulo math também disponibiliza a constante math.pi (π ≈ 3.14159...) e a função abs() — que nem precisa de import — retorna o valor absoluto (sem sinal negativo) de um número.
pi e abs()
import math

print(math.pi)        # 3.141592653589793
print(abs(-15))        # 15  — valor absoluto
print(abs(-3.7))       # 3.7

# Área de um círculo: π × r²
raio = 5
print(math.pi * raio ** 2)  # 78.539...
Resumo das funções do módulo math sqrt(n) — raiz quadrada  |  ceil(n) — arredonda para cima  |  floor(n) — arredonda para baixo  |  trunc(n) — parte inteira  |  abs(n) — valor absoluto (sem import)

Módulo 08
Módulo random
Gerando valores aleatórios e trabalhando com listas em Python.
Conceito
O módulo random permite gerar números aleatórios, sortear itens e embaralhar listas. As funções mais usadas são randint(), choice() e shuffle().
O que é uma lista?
Uma lista é uma forma de guardar vários valores em uma única variável. Em vez de criar aluno1, aluno2, aluno3... você guarda tudo dentro de colchetes separado por vírgulas. Cada valor ocupa uma posição que começa em 0.
Listas — introdução
frutas = ['maçã', 'banana', 'uva']

print(frutas[0])   # maçã   — primeiro elemento (índice 0)
print(frutas[2])   # uva    — terceiro elemento (índice 2)
print(len(frutas)) # 3      — quantidade de itens
Nota Neste módulo vamos usar listas de forma introdutória — apenas criar e passar para as funções do random. O estudo completo de listas (adicionar, remover, ordenar, percorrer) acontece no Módulo 14.
Módulo random
Com a lista criada, as funções do módulo random permitem sortear e embaralhar seus elementos de forma simples:
Módulo random
from random import randint, choice, shuffle

# Número aleatório entre 1 e 10
print(randint(1, 10))

# Sortear um item de uma lista
frutas = ['maçã', 'banana', 'uva']
print(choice(frutas))

# Embaralhar a lista (modifica a original)
shuffle(frutas)
print(frutas)

Módulo 09
Conversão de unidades
Trabalhando com múltiplas variáveis derivadas de um único valor de entrada.
Conceito
Muitas vezes queremos apresentar o mesmo dado em formatos diferentes. Para isso, calculamos cada resultado a partir do valor original e armazenamos em variáveis separadas, depois exibimos tudo de forma organizada.

print('{}m = {}km = {:.0f}cm = {:.0f}mm'.format(
    metros, km, cm, mm
))

Módulo 12
Laços de repetição: for / while
Execute blocos de código múltiplas vezes de forma automática.
O laço for
O for em Python percorre uma sequência de valores. Combinado com range(), você controla quantas vezes o bloco se repete e com quais valores.
for com range()
# range(5) gera: 0, 1, 2, 3, 4
for i in range(5):
    print('Iteração', i)

# range(1, 6) gera: 1, 2, 3, 4, 5
for i in range(1, 6):
    print(i)

# range(0, 11, 2) gera: 0, 2, 4, 6, 8, 10
for i in range(0, 11, 2):
    print(i)
range(início, fim, passo) range(5) — de 0 até 4  |  range(1,6) — de 1 até 5  |  range(0,10,2) — pares de 0 a 8
O laço while
O while repete o bloco enquanto uma condição for verdadeira. É útil quando você não sabe de antemão quantas repetições serão necessárias. Cuidado: se a condição nunca ficar falsa, o programa entra em loop infinito!
while
contador = 1
while contador <= 5:
    print('Contagem:', contador)
    contador = contador + 1  # ou: contador += 1
Atenção Sempre garanta que a variável de controle do while seja atualizada dentro do loop, caso contrário o programa nunca para.

Módulo 13
Funções com def
Agrupe blocos de código reutilizáveis em funções com parâmetros e retorno.
Conceito
Uma função é um bloco de código nomeado que pode ser executado ("chamado") quantas vezes quiser. Funções evitam repetição de código e tornam o programa mais organizado.
Definimos uma função com a palavra-chave def, seguida do nome e dos parâmetros entre parênteses. O return devolve um valor para quem chamou a função.
Definindo e chamando funções

    print('Olá, {}!'.format(nome))

saudacao('Ana')    # saída: Olá, Ana!

Função com return
def soma(a, b):
    return a + b

resultado = soma(10, 5)
print(resultado)  # saída: 15
Boas práticas Nomeie suas funções com verbos que descrevam o que fazem: calcular_area(), converter_temperatura(), verificar_par(). Use underline para separar palavras.
Parâmetros com valor padrão
Parâmetros podem ter um valor padrão. Se o chamador não passar esse argumento, o valor padrão é usado automaticamente.
Valor padrão
def potencia(base, expoente=2):
    return base ** expoente

print(potencia(4))     # 16  (expoente padrão = 2)
print(potencia(2, 10)) # 1024

Módulo 14
Listas
Armazene e manipule coleções de valores em uma única variável.
Conceito
Uma lista é uma coleção ordenada de valores. Cada valor ocupa uma posição (índice) que começa em 0. Você pode guardar inteiros, floats, strings ou até outras listas dentro de uma lista.
Criando e acessando listas
frutas = ['maçã', 'banana', 'laranja']

print(frutas[0])   # maçã   (primeiro elemento)
print(frutas[2])   # laranja (terceiro elemento)
print(frutas[-1])  # laranja (último elemento)
print(len(frutas)) # 3       (quantidade de itens)
Índices negativos lista[-1] acessa o último elemento, lista[-2] o penúltimo, e assim por diante. Muito útil para acessar o final sem precisar saber o tamanho exato da lista.
Métodos essenciais
Listas possuem métodos que permitem adicionar, remover e reorganizar elementos de forma simples.
Métodos de lista
numeros = [3, 1, 4, 1, 5]

numeros.append(9)     # adiciona 9 ao final → [3,1,4,1,5,9]
numeros.remove(1)     # remove a primeira ocorrência de 1 → [3,4,1,5,9]
numeros.sort()        # ordena em ordem crescente → [1,3,4,5,9]
print(sum(numeros))   # soma todos os elementos → 22
print(max(numeros))   # maior valor → 9
print(min(numeros))   # menor valor → 1
Percorrendo uma lista com for
O for percorre cada elemento de uma lista diretamente, sem precisar de índices. Use enumerate() quando precisar do índice junto com o valor.
Iterando sobre listas
notas = [8.5, 7.0, 9.2, 6.8]

# Percorrendo os valores
for nota in notas:
    print(nota)

# Percorrendo com índice
for i, nota in enumerate(notas):
    print('Aluno {}: {}'.format(i + 1, nota))

Módulo 15
Strings em profundidade
Manipule texto com fatiamento, métodos e busca em strings.
Strings são sequências
Em Python, uma string é uma sequência de caracteres — assim como uma lista, cada caractere tem um índice que começa em 0. Isso significa que você pode fatiar, iterar e medir strings da mesma forma que listas.
Indexação e fatiamento
palavra = 'Python'

print(palavra[0])      # P   (primeiro caractere)
print(palavra[-1])     # n   (último caractere)
print(palavra[0:3])    # Pyt (índice 0 até 2)
print(palavra[::-1])   # nohtyP (invertida)
print(len(palavra))    # 6
Fatiamento: texto[início:fim:passo] 'Python'[1:4] → 'yth'  |  'Python'[::-1] → 'nohtyP'  |  'Python'[::2] → 'Pto'
Métodos essenciais
Strings possuem dezenas de métodos embutidos. Os mais usados transformam capitalização, removem espaços, substituem trechos e dividem o texto em pedaços.
Métodos de string
texto = '  Olá, Mundo!  '

print(texto.strip())          # 'Olá, Mundo!'  (remove espaços das bordas)
print(texto.upper())          # '  OLÁ, MUNDO!  '
print(texto.lower())          # '  olá, mundo!  '
print(texto.replace('Mundo', 'Python'))  # '  Olá, Python!  '
print(texto.strip().split(', '))         # ['Olá', 'Mundo!']
print(texto.strip().startswith('Olá'))  # True
print(texto.strip().count('o'))          # 1
Busca e verificação
Use in para verificar se um trecho existe dentro de uma string, e find() para descobrir em qual posição ele aparece.
Busca em strings
email = 'usuario@exemplo.com'

print('@' in email)            # True
print(email.find('@'))         # 7  (posição do @)
print(email.endswith('.com'))  # True
print(email.isdigit())         # False
print('123'.isdigit())          # True

frase = input('Digite uma frase: ')

        vogais += 1

print('Frase limpa: {}'.format(limpa))
print('Invertida: {}'.format(invertida))

Validador de senha
Funções + strings + múltiplas verificações
Crie a função validar_senha(senha) que retorna True se a senha atender a todos os critérios abaixo, ou False caso contrário. Exiba quais critérios foram aprovados e quais falharam.

Critérios: mínimo de 8 caracteres, pelo menos uma letra maiúscula, pelo menos um dígito (isdigit()), pelo menos um caractere especial de !@#$%.
● Carregando Python...
▶ Saída
Solução
def validar_senha(senha):
    tem_tamanho   = len(senha) >= 8
    tem_maiuscula = False
    tem_digito    = False
    tem_especial  = False
    for c in senha:
        if c.isupper():
            tem_maiuscula = True
        if c.isdigit():
            tem_digito = True
        if c in '!@#$%':
            tem_especial = True

    def status(ok):
        return '✓' if ok else '✗'

    print('[{}] Mínimo 8 caracteres'.format(status(tem_tamanho)))
    print('[{}] Letra maiúscula'.format(status(tem_maiuscula)))
    print('[{}] Dígito numérico'.format(status(tem_digito)))
    print('[{}] Caractere especial (!@#$%)'.format(status(tem_especial)))

    return tem_tamanho and tem_maiuscula and tem_digito and tem_especial

senha = input('Digite uma senha: ')
if validar_senha(senha):
    print('Senha forte!')
else:
    print('Senha fraca. Corrija os itens marcados com ✗.')

Validador de e-mail
find(), startswith(), endswith() e in
Leia um endereço de e-mail e verifique: se contém @ (use in), a posição do @ (use find()), se termina com .com ou .br (use endswith()). Se o @ for encontrado, exiba o nome de usuário (parte antes do @) e o domínio (parte depois). Ao final informe se o e-mail é válido ou não.
● Carregando Python...
▶ Saída
Solução
email = input('Digite seu e-mail: ').strip().lower()

tem_arroba  = '@' in email
pos_arroba  = email.find('@')
termina_ok  = email.endswith('.com') or email.endswith('.com.br') or email.endswith('.br')

print('Contém @: {}'.format(tem_arroba))
print('Posição do @: {}'.format(pos_arroba))

if tem_arroba:
    usuario = email[:pos_arroba]
    dominio = email[pos_arroba + 1:]
    print('Usuário: {}'.format(usuario))
    print('Domínio: {}'.format(dominio))
    print('Termina com .com/.br: {}'.format(termina_ok))

if tem_arroba and termina_ok:
    print('E-mail válido!')
else:
    print('E-mail inválido.')
Módulo 16
Dicionários
Armazene dados em pares chave–valor para acesso rápido e organizado.
Conceito
Um dicionário armazena pares chave: valor. Em vez de acessar dados por índice numérico (como em listas), você acessa pelo nome da chave — o que torna o código muito mais legível quando os dados têm significado próprio.
Criando e acessando dicionários
aluno = {
    'nome': 'Ana',
    'idade': 20,
    'nota': 8.5
}

print(aluno['nome'])   # Ana
print(aluno['nota'])   # 8.5

# Adicionando e atualizando chaves
aluno['curso'] = 'Engenharia'
aluno['nota'] = 9.0    # sobrescreve o valor anterior

print(aluno)
dict.get(chave, padrão) Use aluno.get('email', 'não informado') para acessar uma chave que pode não existir — assim você evita o erro KeyError e recebe um valor padrão caso a chave esteja ausente.
Métodos essenciais
Os três métodos mais usados para percorrer um dicionário são keys(), values() e items(). Use-os com for para iterar sobre chaves, valores ou ambos ao mesmo tempo.
Iterando sobre dicionários
produto = {'nome': 'Teclado', 'preco': 149.90, 'estoque': 12}

for chave in produto.keys():
    print(chave)                    # nome, preco, estoque

for valor in produto.values():
    print(valor)                    # Teclado, 149.9, 12

for chave, valor in produto.items():
    print('{}: {}'.format(chave, valor))
Verificando e removendo chaves
in, pop e len
config = {'tema': 'escuro', 'idioma': 'pt-BR', 'fonte': 14}

print('tema' in config)           # True
print('volume' in config)         # False
print(len(config))                 # 3

config.pop('fonte')               # remove a chave 'fonte'
print(config)                      # {'tema': 'escuro', 'idioma': 'pt-BR'}

Contador de frequência
Dicionário como contador
Leia uma frase e conte quantas vezes cada palavra aparece (ignore maiúsculas/minúsculas). Use um dicionário onde a chave é a palavra e o valor é a contagem. Exiba as palavras em ordem alfabética com suas frequências.
● Carregando Python...
▶ Saída
Solução
frase = input('Digite uma frase: ').lower()
palavras = frase.split()

freq = {}
for palavra in palavras:
    freq[palavra] = freq.get(palavra, 0) + 1

for palavra in sorted(freq):
    print('{}: {}'.format(palavra, freq[palavra]))

Cardápio de restaurante
Lista de dicionários + cálculo de total
Crie uma lista com 4 pratos, onde cada prato é um dicionário com as chaves 'nome' e 'preco'. Os dados devem ser lidos do usuário. Exiba o cardápio formatado e, ao final, o prato mais barato, o mais caro e a média de preços.
● Carregando Python...
▶ Saída
Solução
cardapio = []
for i in range(4):
    nome  = input('Nome do prato {}: '.format(i + 1))
    preco = float(input('Preço: R$ '))
    cardapio.append({'nome': nome, 'preco': preco})

print('\n--- CARDÁPIO ---')
for prato in cardapio:
    print('{:<20} R$ {:.2f}'.format(prato['nome'], prato['preco']))

mais_barato = cardapio[0]
mais_caro   = cardapio[0]
total = 0
for prato in cardapio:
    total += prato['preco']
    if prato['preco'] < mais_barato['preco']:
        mais_barato = prato
    if prato['preco'] > mais_caro['preco']:
        mais_caro = prato
media = total / len(cardapio)

print('Mais barato: {} (R$ {:.2f})'.format(mais_barato['nome'], mais_barato['preco']))
print('Mais caro:   {} (R$ {:.2f})'.format(mais_caro['nome'], mais_caro['preco']))
print('Média: R$ {:.2f}'.format(media))

Agenda de contatos
Dicionário de dicionários + busca
Crie uma agenda onde a chave é o nome e o valor é um dicionário com 'telefone' e 'email'. Cadastre 3 contatos lidos do usuário. Depois leia um nome para buscar: se existir, exiba os dados; se não, exiba "Contato não encontrado." Em seguida, leia um nome para remover da agenda usando pop() (ou informe que não existe). Ao final, liste todos os contatos restantes em ordem alfabética.
● Carregando Python...
▶ Saída
Solução
agenda = {}
for i in range(3):
    nome  = input('Nome do contato {}: '.format(i + 1)).strip().title()
    tel   = input('Telefone: ')
    email = input('Email: ')
    agenda[nome] = {'telefone': tel, 'email': email}

busca = input('\nBuscar contato: ').strip().title()
if busca in agenda:
    dados = agenda[busca]
    print('Nome: {}'.format(busca))
    print('Telefone: {}'.format(dados['telefone']))
    print('Email: {}'.format(dados['email']))
else:
    print('Contato não encontrado.')

remover = input('\nRemover contato: ').strip().title()
if remover in agenda:
    agenda.pop(remover)
    print('{} removido da agenda.'.format(remover))
else:
    print('Contato {} não encontrado.'.format(remover))

print('\n--- TODOS OS CONTATOS ---')
for nome in sorted(agenda):
    d = agenda[nome]
    print('{} | {} | {}'.format(nome, d['telefone'], d['email']))
