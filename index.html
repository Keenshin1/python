Módulo 01
Saída de dados com print()
A primeira coisa que todo programador aprende: exibir informações na tela.
Conceito
Em Python, usamos a função print() para exibir qualquer coisa na tela. Você passa o que quer exibir dentro dos parênteses.
Texto (chamado de string) precisa estar entre aspas — pode ser simples 'texto' ou duplas "texto".
Python
print('Olá Mundo!')
print("Python é incrível")
print(42)
print(3.14)
Saída do programa Olá Mundo!
Python é incrível
42
3.14
Dica Você pode usar \n dentro de uma string para pular linha: print('linha1\nlinha2')
Variáveis
Uma variável é um nome que guarda um valor na memória do programa. Pense nela como uma caixinha com um rótulo: você dá um nome à caixinha e coloca algo dentro.
Para criar uma variável, use o sinal = — que aqui significa "recebe", não igualdade matemática. O nome fica à esquerda e o valor à direita.
Python
nome = 'Tainá'        # a variável 'nome' guarda o texto 'Tainá'
idade = 28         # a variável 'idade' guarda o número 28
print(nome)        # exibe: Tainá
print(idade)       # exibe: 28
Atenção Nomes de variáveis não podem ter espaços nem começar com número. Use letras, números e underline (o traço embaixo _): nome_completo, nota1.

Módulo 02
Entrada de dados com input()
Como fazer o programa conversar com o usuário e capturar o que ele digita.
Conceito
A função input() pausa o programa e espera o usuário digitar algo. O que for digitado é retornado como uma string e pode ser guardado em uma variável.
Dentro dos parênteses de input() você pode colocar uma mensagem para orientar o usuário — isso é chamado de prompt.
Python
nome = input('Qual é o seu nome? ')
print('Olá, ' + nome + '!')
Atenção input() sempre retorna uma string, mesmo que o usuário digite um número. Para usar como número, é preciso converter com int() ou float().
Para combinar textos e variáveis no print(), use o operador + para concatenar (juntar) strings. Qualquer parte que não seja variável precisa estar entre aspas.
Concatenação com +
nome = input('Qual é o seu nome? ')
cidade = input('De onde você é? ')
print('Olá, ' + nome + '! Você é de ' + cidade + '.')

Módulo 03
Operações matemáticas
Python como calculadora: soma, subtração, multiplicação, divisão e muito mais.
Conceito
Python suporta todas as operações matemáticas básicas. Para trabalhar com números digitados pelo usuário, precisamos converter a string que input() retorna usando int() para inteiros ou float() para decimais.
int() e float() são funções de conversão. Você passa um valor dentro dos parênteses e recebe o mesmo valor em outro tipo:
Conversão de tipo
int('10')       # transforma a string '10' no número 10
float('3.14')   # transforma a string '3.14' no número 3.14
str(42)         # transforma o número 42 na string '42'
Na prática, combinamos int() com input() em uma só linha — o resultado de input() vai direto para dentro de int():
Operadores básicos
+ - * / // % **
Conversão de tipo
int() · float() · str()
Python
n1 = int(input('Primeiro número: '))
n2 = int(input('Segundo número: '))

print(n1 + n2)   # Soma
print(n1 - n2)   # Subtração
print(n1 * n2)   # Multiplicação
print(n1 / n2)   # Divisão real
print(n1 ** 2)   # Potência
print(n1 % n2)   # Resto da divisão

Módulo 04
Tipos primitivos e métodos de string
Entendendo os tipos de dados e explorando as capacidades das strings em Python.
Conceito
Todo valor em Python tem um tipo. Os principais são: str (texto), int (inteiro), float (decimal) e bool (verdadeiro/falso). Use type() para descobrir o tipo de qualquer valor.
Strings possuem métodos — funções que já vêm embutidas em todo texto. Pense assim: assim como um canivete suíço tem ferramentas dentro dele, uma string carrega suas próprias ferramentas. Para usar, escreva o nome da variável, um ponto, e o nome da ferramenta: variavel.metodo().
Métodos de string
texto = 'Python'

print(type(texto))          # descobre o tipo → <class 'str'>

# Métodos de verificação — retornam True ou False
print(texto.isupper())      # está tudo em MAIÚSCULAS? → False
print(texto.islower())      # está tudo em minúsculas? → False
print(texto.isalpha())      # contém só letras (sem número)? → True
print(texto.isnumeric())    # contém só números? → False
print(texto.isalnum())      # contém só letras e/ou números? → True
print(texto.istitle())      # cada palavra começa com maiúscula? → True

# Métodos de transformação — retornam o texto modificado
print(texto.upper())        # converte tudo para MAIÚSCULAS → PYTHON
print(texto.lower())        # converte tudo para minúsculas → python
print(texto.title())        # primeira letra de cada palavra em maiúscula → Python

# len() conta o número de caracteres
print(len(texto))           # quantos caracteres tem? → 6

Módulo 05
Formatação de saída
Controlando casas decimais, alinhamento e como os dados aparecem na tela.
Conceito
O método .format() é uma forma mais organizada de montar strings com variáveis. Em vez de usar + e str(), você escreve {} dentro da string onde quer inserir um valor, e passa os valores dentro de .format():
Básico do .format()
nome = 'Tainá'
idade = 28

# com +  e str() — jeito antigo
print('Olá ' + nome + ', você tem ' + str(idade) + ' anos.')

# com .format() — mais limpo
print('Olá {}, você tem {} anos.'.format(nome, idade))
Dentro das chaves {} você também pode usar especificadores para controlar como os números são exibidos — útil especialmente para decimais:
Exemplos de formatação
n = 3.14159
nota = 8.5

print('{:.2f}'.format(n))      # 3.14 — 2 decimais
print('{:.0f}'.format(n))      # 3 — sem decimais
print('{:10.2f}'.format(n))    # "      3.14" — largura 10
print('{:1f}'.format(nota))    # 8.5

# f-strings (Python 3.6+) — alternativa moderna
print(f'Nota: {nota:.1f}')
Separador de linhas Você pode repetir um caractere com *: print(20 * '=') imprime 20 sinais de igual seguidos. Útil para criar bordas visuais.

Módulo 06
Importações e o módulo math
Usando bibliotecas prontas para cálculos matemáticos mais avançados.
Conceito
Python vem com módulos prontos que adicionam funcionalidades extras. Importamos com a palavra-chave import. O módulo math oferece funções matemáticas como raiz quadrada, seno, cosseno, tangente, etc.
Formas de importar
import math
math.sqrt(16)     # 4.0

from math import sqrt, trunc, hypot
sqrt(16)          # 4.0 (sem prefixo math.)
trunc(6.9)        # 6 (parte inteira)

from math import sin, cos, tan, radians
sin(radians(90))  # 1.0 (ângulo em graus → radianos)
Atenção com ângulos As funções trigonométricas do Python trabalham em radianos. Para passar graus, converta com radians(ângulo).

Módulo 08
Módulo random
Gerando valores aleatórios e trabalhando com listas em Python.
Conceito
O módulo random permite gerar números aleatórios, sortear itens e embaralhar listas. As funções mais usadas são randint(), choice() e shuffle().
Listas são criadas com colchetes e armazenam múltiplos valores: [item1, item2, item3]. Você pode adicionar variáveis a uma lista e operar sobre elas.
Módulo random
from random import randint, choice, shuffle

# Número aleatório entre 1 e 10
print(randint(1, 10))

# Sortear um item de uma lista
frutas = ['maçã', 'banana', 'uva']
print(choice(frutas))

# Embaralhar a lista (modifica a original)
shuffle(frutas)
print(frutas)

Módulo 09
Conversão de unidades
Trabalhando com múltiplas variáveis derivadas de um único valor de entrada.
Conceito
Muitas vezes queremos apresentar o mesmo dado em formatos diferentes. Para isso, calculamos cada resultado a partir do valor original e armazenamos em variáveis separadas, depois exibimos tudo de forma organizada.
Exemplo — metros para outras unidades
metros = float(input('Distância em metros: '))

km = metros / 1000
cm = metros * 100
mm = metros * 1000

print('{}m = {}km = {:.0f}cm = {:.0f}mm'.format(
    metros, km, cm, mm
))
